# Глоссарий

Добавьте сюда термины и сокращения, которые встречаются в регламентах.

## Пример оформления
- **СЛА** — целевой срок выполнения (Service Level Agreement).
- **Инициатор** — сотрудник/подразделение, которое запускает процесс.
- **Ответственный** — роль, которая выполняет шаг и отвечает за результат.
